# 模板管理
> V-box云组态平台提供模板管理功能，可将设计好的组态画面模板保存起来，以供复用。在设计工程的组态画面时，导入模板即可。
 
### 管理模板
1. 新增模板
点击按钮，可新增一个模板。
2. 编辑模板
点击按钮，可编辑该模板的名称和分辨率。
3. 删除模板
点击按钮，可删除该模板。
4. 搜索模板
在模板列表左上角的搜索框中输入模板名称，或组合模板分辨率进行搜索。

### 模板设计
点击模板列表中某个模板后面的 按钮，可进入V-BOX云组态平台的组态编辑器页面。可在编辑器页面对该模板进行组态画面的设计。组态画面的详细设计方法，参阅第六章工程组态设计。

##### 模板管理注意事项
1. 工程的组态画面中仅可导入相同分辨率的模板
2. 模板设计仅支持单画面设计
3. 模板画面不支持导入导出、预览、编译、导入其他模板、绑定监控点功能
